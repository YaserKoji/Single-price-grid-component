# Frontend Mentor - Single Price Grid Component Solution

This is my clean and responsive solution to the Single Price Grid Component challenge on Frontend Mentor.

## My Process

### Built with

- Semantic HTML5 markup
- CSS Custom Properties (Variables)
- CSS Grid (for desktop layout)
- Flexbox (for centering the card)
- Media Queries for responsiveness

### How I built it

1. **HTML Architecture First**: I started by writing clean, semantic HTML, ensuring the card components are structured properly using a `<main>` container and `<ul>` lists for best accessibility practices.
2. **Sequential & Organized CSS**: Next, I styled the project sequentially. I used highly organized class names and targeted them cleanly in CSS to keep the code neat, readable, and easy to maintain.
3. **Grid Layout**: I implemented CSS Grid to perfectly split the sections on desktop screens and made it fully responsive for mobile devices.
